# Nhạc nền thiệp cưới

Đặt file nhạc nền vào thư mục này với tên chính xác:

```
wedding-song.mp3
```

Trang web đã trỏ sẵn thẻ `<audio>` tới `audio/wedding-song.mp3`.
Khi chưa có file, nút nhạc tự động ẩn — thêm file vào là nút hiện ra, không cần sửa code.

Gợi ý nguồn nhạc free-license:
- https://pixabay.com/music/search/wedding/ (Pixabay License — dùng miễn phí)
- https://incompetech.com (Kevin MacLeod, CC-BY — cần ghi nguồn)

Lưu ý: nhạc có bản quyền (nhạc thị trường) không nên dùng khi đưa web lên mạng công khai.
